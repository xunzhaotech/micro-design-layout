# MicroDesign

# 说明

MicroDesign是一款基于微服务的后端管理系统架构,致力于打造高效、规范、统一的企业级后端管理平台，采用全新的Vue3.0open in new window新版API之Composition API搭建，UI采用开源Ant Design Vueopen in new window框架，致力打造服务企业的微服务架构后端管理框架。
